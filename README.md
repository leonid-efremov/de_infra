# de_infra
Infrastructure for running DE pipelines locally

Первый запуск на Windows:
1. Ставим и запускаем Docker Desktop
2. Обновляем WSL (cmd: wsl --update)
3. Запускаем из папки проекта (cmd: docker compose up)
